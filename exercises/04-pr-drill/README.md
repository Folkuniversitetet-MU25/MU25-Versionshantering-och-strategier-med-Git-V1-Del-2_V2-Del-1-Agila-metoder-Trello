# 04 – PR-flöde (mini) + review + länka kort

**Mål:** Träna PR-disciplin kopplat till brädan.

## Steg
1) Välj **1 liten story** från “Selected for Sprint”.
2) Skapa **feature-branch** i ditt repo (ex: `feat/login-button`).
3) Gör **minst 1 commit** med **bra meddelande** (ex: `feat: add login button`).
4) Öppna **PR**:
   - Följ **PR-policy** (rubrik, beskrivning varför/hur, ev. bild)
   - **Länka Trello-kortet** i PR-beskrivningen
5) Få **1 review** (klasskamrat), adressera feedback, **mergea** (Squash & merge).
6) **Stäng kortet** i Trello när DoD uppfyllt.

## Checklista
- [ ] PR följer policy och länkar Trello-kort
- [ ] Minst 1 review gjord
- [ ] Kort stängt vid merge (flyttat till Done)
